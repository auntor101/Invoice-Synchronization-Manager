# Invoice Synchronization Manager

Enterprise invoice management system with offline capability and OCR processing.

## Features

- Scan invoices using camera
- Upload multiple invoice files
- Automatic OCR processing with confidence scoring
- Offline mode with sync queue
- Real-time search and filtering
- Export functionality
- Responsive design for mobile and desktop

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

This project is configured for automatic deployment on Netlify. Simply connect your repository to Netlify and it will build and deploy automatically.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons
